### rc/v1.6.2.3/20240625

功能说明：

诊断分析，新增波形列表分析方式，目前开发支持具体功能点

- 点击左侧设备树中的机组、部件、测点位置，右上主图视区显示对应的波形列表

- 波形列表支持勾选、折叠等，默认显示 7 天数据，支持转速设置

- 波形分析勾选后的波形数据，切换部件或者机组后，支持保留上次的数据，如果当前列表中存在上次的分析数据

- 波形列表为主视图，新增的图谱组件更改图谱类型不支持特征值相关

- 点击设备树，右侧视图趋势与波形列表切换后，波形图谱数据清空

诊断分析特征值趋势和波形索引支持同一测点不同波形定义的采集波形显示：

- 趋势分析，如果存在双波形，点击趋势点后，相关波形图谱显示所有的波形数据（波形命名中以采集频率进行区分）

- 波形索引：采集时间点弹出的测量位置信息命名中，增加采集频率进行区分

### rc/v1.6.3.1/20240815

新增功能说明（诊断分析页面开发第二个版本）：

诊断分析页面新版本功能调整及开发：

- 去掉波形索引分析

- 左侧功能区：设备树增加多选框；集成了日期、转速等条件筛选；日期面板增加显示“所勾选设备”是否有数据，进行标记。

- 右侧主图谱（右上）功能区，目前显示两种分析方式：趋势分析和波形列表分析，可进行切换；其中趋势分析对设备树中选中的特征值进行数据响应，波形列表对设备树中选中或者半选中的测量位置进行数据响应

- 交互：
  左侧功能区改变，点击“数据查询”，右侧图谱功能区图谱重置重新请求数据
  主图谱功能区提供基础分析数据，支持数据的添加选中，取消选中操作，辅助图谱进行数据响应
