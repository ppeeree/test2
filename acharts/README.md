## 2025/11/25

### 优化

在 Bladex 项目（或任何基于 Webpack / Vite / Rollup 的前端项目）里，如果把「自己封装的组件库」通过 npm link（或 yarn link / pnpm link）链进主工程，极易出现 “第三方依赖被重复打包” 的问题。
本质原因是：
link 之后，组件库有自己独立的 node_modules，Webpack 会把它当成两个完全独立的包，于是把 echarts、element 又打了一遍。

解决方案：

1. 把组件库的 node_modules 里的依赖，都移到主工程的 node_modules 里，这样 Webpack 就不会重复打包了。把组件库声明为 peerDependencies，这样主工程在安装组件库时，会自动安装这些依赖，一般主工程也存在这些依赖，所以不会报错。
   在组件库 package.json 里 不要把 echarts 等写进 dependencies，而是写到
   peerDependencies + devDependencies

2. 如果组件库和主工程 都打算用 CDN 引入 Vue / Element，可直接 externals
   // webpack
   module.exports = {
   externals: {
   vue: 'Vue',
   'element-plus': 'ElementPlus'
   }
   }

## 安装

```
npm install acharts
```

## 导入

```
import {TimeDomainChart，***} from 'acharts'
TimeDomainChart：时域波形
 其他图谱类名：
  EvTrendLine, // 特征值趋势
  ScatterChart, // 相关性分析-散点图
  HistogramChart, // 分布--柱状区间统计图
  DipAngleChart, // 倾覆分析
  WaterFallChart, // 瀑布图
  SpectrumChart, // 频谱图
  WaveFormIndexChart, // 波形索引图
  RPMWave // 转速波形
```

## 使用

```
let dom = document.getElementById(idname)
let mychart = echarts.init(dom)
let instance = new TimeDomainChart(
            mychart, // echarts实例
            { ...chartData }, // 数据
            'light' // 主题，也可不传，默认白色
          )
```

## 数据格式

```
    {
    dimensions: ['时间(s)', 'm/s^2'], //X,Y轴物理量固定不变的图谱，eg：倾覆分析 [X轴物理量名称或者单位，Y轴物理量名称或者单位]
    titleText: '时域分析', // 图谱标题
    yAxisType: 'value', // 图谱Y轴的数据类型
    xAxisType: 'value', // 图谱X轴的数据类型
    xInterval：0 || null,  // X轴的时间间隔，可不传，也可传0或者null
    data: [{ // 每条图谱数据数组，包含：名称，id，数据源
      name: '#001_2023-04-13 18:00:02', // name 唯一
      id: '000001_2023-04-13 18:00:02', // id 必须唯一
      source: [[0.1, 2], [0.11, 0.21], [0.22, 0.8]], // [xValue, yValue]
      dimensions：[X轴物理量名称或者单位，Y轴物理量名称或者单位], // 多特征值，特征值物理量不同时，使用堆叠方式，需要显示Y轴物理量
      xInterval:Number||null // x轴间隔，时域波形一般会使用，
    }],
    }
```

- ### 核心功能
  - **游标分析**
    1. 添加：点击游标类型，会给当前图谱每条数据进行添加
    2. 删除：右键点击某条游标线，可进行单组游标删除
    3. 清空
    4. 游标线鼠标拖动
    5. 鼠标左键点击某条游标线后，可用键盘左右键控制游标线平移
    6. 游标信息展示，右上角图标可进行点击查看
    7. 游标线鼠标划上响应，查看当前游标线的坐标信息
    8. 多曲线时，曲线的堆叠方式可进行协助分析
  - **趋势图点击操作**
    1. 鼠标单击，选中，单选互斥
    2. ctrl+鼠标单击，多点选中
